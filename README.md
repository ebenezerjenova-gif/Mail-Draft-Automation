# 📧 Automated Mail Draft Workflow using n8n

This project demonstrates a simple yet effective workflow automation built using **n8n**. The workflow is designed to automatically generate personalized email drafts by fetching user data from a Google Sheet.

## 🚀 Features

* 📥 Fetches recipient details directly from **Google Sheets**
* ✉️ Automatically generates **personalized email drafts**
* ⚡ Reduces manual effort in email drafting
* 🔄 Demonstrates seamless integration between tools using n8n

## 🛠️ Tech Stack

* **n8n** (Workflow Automation)
* **Google Sheets API**
* Email Draft Automation Logic

## 📌 Workflow Overview

1. Connects to a Google Sheet containing recipient data
2. Reads user details such as name and email
3. Processes the data and generates customized email content
4. Creates ready-to-use email drafts for review and sending
